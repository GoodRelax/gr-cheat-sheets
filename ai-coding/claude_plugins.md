## ③ 遅延ロードツール（`tool_search` で読み込み後に使用可能）

> `tool_search` ツールで検索・ロードしてから使用すること。  
> すべて **Claude in Chrome** カテゴリに属する。

| # | ツール名 | 概要 |
|---|----------|------|
| 1 | `computer` | マウス・キーボード操作でブラウザを操作し、スクリーンショットを撮る |
| 2 | `file_upload` | ローカルファイルシステムからファイルをアップロードする |
| 3 | `find` | 自然言語でページ上の要素を検索する |
| 4 | `form_input` | `read_page` が返した要素 ID を使ってフォームに値をセットする |
| 5 | `get_page_text` | 現在のページからテキストを抽出する（記事コンテンツ優先） |
| 6 | `gif_creator` | ブラウザ自動化セッションの GIF 録画・エクスポートを管理する |
| 7 | `javascript_tool` | 現在のページのコンテキストで JavaScript を実行する |
| 8 | `navigate` | URL に移動する、またはブラウザ履歴を前後に移動する |
| 9 | `read_console_messages` | ブラウザのコンソールメッセージ（log / error / warn 等）を読み取る |
| 10 | `read_network_requests` | HTTP ネットワークリクエスト（XHR / Fetch / 画像等）を読み取る |
| 11 | `read_page` | ページ上の要素をアクセシビリティツリーとして取得する |
| 12 | `resize_window` | ブラウザウィンドウを指定サイズにリサイズする |
| 13 | `shortcuts_execute` | ショートカット／ワークフローをサイドパネルウィンドウで実行する |
| 14 | `shortcuts_list` | 利用可能なショートカット・ワークフローを一覧表示する |
| 15 | `switch_browser` | ブラウザ自動化に使用する Chrome ブラウザを切り替える |
| 16 | `tabs_close_mcp` | MCP タブグループ内のタブを ID で閉じる |
| 17 | `tabs_context_mcp` | 現在の MCP タブグループのコンテキスト情報を取得する |
| 18 | `tabs_create_mcp` | MCP タブグループに新しい空のタブを作成する |
| 19 | `upload_image` | キャプチャしたスクリーンショットやユーザーがアップロードした画像をファイル入力に渡す |
